# v-wave-surfer

Vue.js components for observing some articulatory data.

## Introduction

このツールは 舌運動計測装置 WAVE で収録された調音運動データを観察するための
フロントエンドツールになります．

- vue.js
- vuetify.js
- vue-threejs
- wavesufer.js
